# Ethical-Data-Analysis-using-SAS-Viya

This hands-on workshop explores predictive modeling and ethical decision-making using a guided scenario in SAS®  Model Studio. You’ll learn how bias and fairness issues can arise in real-world analytics, and how SAS®  Model Studio’s built-in fairness and bias assessment tools help identify and mitigate them. Emphasis will be placed on refining models to balance predictive performance with equitable outcomes.
